# RipAdbaisor

Este proyecto es una aplicación en Java que permite añadir, editar, mostrar y eliminar elementos que tienen una puntuación del 1 al 5.
La aplicación usa Java Swing para pedir los datos y mostrar la información porque el cliente no quiere usar la consola.

Los datos se guardan en una lista (ArrayList) mientras el programa está abierto, pero no se guardan entre ejecuciones.

¿Qué puede hacer la aplicación?
Añadir elemento  
Pide nombre, ciudad, tipo, precio medio y puntuación.

Editar elemento  
Permite cambiar los datos de un elemento usando su índice.

Mostrar elementos  
Enseña todos los elementos ordenados por puntuación de mayor a menor.

Eliminar elemento  
Borra un elemento usando su índice.

Salir  
Cierra el programa.
